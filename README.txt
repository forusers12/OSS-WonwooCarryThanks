0. 개요

본 프로그램은 C라이브러리 함수 교육을 제공하며, 다른 사이트와 달리 직관화된 이미지를 이용하여 조금 더 친밀하게 다가갈 수 있다는 장점이 있다. 기존의 컴퓨터 교육사이트의 경우는 대부분이 딱딱한 이미지로써 사용자에게 다가가지만, 본 페이지는 학생들에게 초점을 맞추어 디자인된 페이지로써 직관화된 레이아웃 구성으로, 쉽게 다가갈 수 있다.


본 프로그램은 회원가입 없이 이용할 수 있으며, 초기화면에서 C를 눌렀을 때 C라이브러리 함수 교육 컨텐츠를 이용할 수 있다. 해당 컨텐츠에서 검색버튼과, 자세히보기 버튼이 제공되는데, 검색버튼은 사용자가 원하는 함수를 빠르게 찾기 위한 기능이며, 자세히보기 버튼은 기본적으로 간단한 설명이 제공되는 것에 덧붙여 상세한 소스 코드의 설명까지 이용할 수 있다.
회원가입을 이용할 시는 댓글란을 사용할 수 있어서 커뮤니티적 개념으로 서로 모르는 개념에 대해 공유할 수 있어서 더 빠른 차원의 이해가 될 수 있는 환경또한 제공한다.


1. 주요파일 역할



<loginserv.php>라는 파일은 세션을 불러올 수 있으며 <login.html>에서
입럭한 id와 password를 가져와 db에서 확인할 수 있는 파일이다.
아이디를 불러오며 관리자인지 아닌지 확인한다.

<signUp.php>파일은 <register.html>에서 입력한 id, password, email를 불러와
db에 연결해 id가 이미 존재하는지 아닌지 확인하고 db의 member테이블에 저장이
되는 파일이다.

<logout.php>파일은 세션 시작했던 것을 종료시키는 파일이다.

<index.php> 메인화면으로 사용자들이 로그인 및 회원가입을 할 수 있고
세션이 실행되며. 세션값이 있으면 id를 불러와 출력할 수 있고
로그아웃을 할 수 있으며 아니면 일반 로그인을 할 수 있게 한다. 또한
홈페이지를 살펴 볼 수 있으며 다른 페이지들로 연결이 되는 메인 페이지다.


<login.html> 파일은 form에서 post형태로 <loginserv.php>에 action을 취하며
form안에 있는 id,password를 보내는 파일이다.

<register.html> 파일은 form에서 post형태로 <signUp.php>에 action을 취하며
form안에 있는 id,password,email를 보내는 파일이다.

<db.php> 파일은 지금 데이터베이스의 이름과 비밀번호를 입력해서 연결하는 파일이다
글로벌 변수를 생성해서 다른 php파일에서 include 해 변수를 이용할 수 있게 함.

<delDB.php>파일은 데이터베이스의 정보들을 가져와 funcName GET으로 받아와 
funcName이 있으면 삭제하는 파일이다.

<managerment.php> 관리자로 로그인 하고 <managerFunc.php>에서 추가를 눌렀을 시 
보여지는 페이지이다. db에서 함수들을 추가하는 파일이다.

<managerDB.php>는 관리자 페이지를 연동하기 위해서 만든 파일이다. db.php를 포함하고
있으며 funcName,head,beginExplain,

<managerFunc.php> 관리자로 로그인이 되면 해당 헤더파일의 목록들을 보여주는 페이지이다.

<searchfuncName.php>는 검색창이 있는 모든 페이지에서 검색을 누르면 연결되는 페이지로
함수이름으로 검색한 결과를 보여주는 페이지이다.

<tables.php>c를 누르면 값을 받아 해당 헤더파일의 목록들을 보여주는 페이지이다.

<viewFunc.php>검색이나 tables에서 자세히 보기 를 눌렀을 경우 해당 함수에 대한 설명을
보여주는 페이지이다
